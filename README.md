## 📖 Sobre o Projeto

O **Gerador de Código CSS com IA** é uma ferramenta que permite criar animações e estilos CSS simplesmente descrevendo o que você imagina. Por trás dos panos, o projeto consome a API da **Groq** com o poderoso modelo **LLaMA 3.3**, que é instruído a responder apenas com código puro (HTML e CSS) baseado na descrição do usuário. O resultado é exibido em tempo real, mostrando tanto o código gerado quanto um preview funcional do que foi criado.

## ✨ Funcionalidades Principais

- ✅ Geração de código CSS personalizado a partir de descrições em texto
- ✅ Pré-visualização em tempo real do código gerado dentro de um iframe
- ✅ Exibição do código-fonte para você copiar e usar
- ✅ Instrução específica para a IA gerar animações 

## 🎯 Objetivos do Projeto

- ✅ Aprender a integrar e consumir APIs de Inteligência Artificial (Groq)
- ✅ Entender como fazer "prompt engineering" para obter resultados precisos da IA
- ✅ Manipular o DOM para exibir código e atualizar um iframe dinamicamente
- ✅ Criar uma experiência de usuário interativa e visualmente atraente
